# עורך טקסט עברי למסך מפוצל

אפליקציה מקומית לעריכת טקסט עברי ב־RTL עם תצוגה זורמת, זום מהיר, הערות שוליים ליד מקום ההפניה, פתיחת קבצי DOCX, טיוטה מקומית והתקנה כאפליקציה.

## פתיחה

פתח את `index.html` בדפדפן.

כדי לאפשר התקנה כאפליקציה ועבודה אופליין, פתח דרך שרת מקומי:

```powershell
python -m http.server 8787
```

ואז היכנס לכתובת:

```text
http://127.0.0.1:8787
```

בדפדפן Chrome או Edge יופיע כפתור "התקנה כאפליקציה" כשאפשר להתקין.

## מה עובד עכשיו

- פתיחת קובץ Word מסוג DOCX.
- הצגת גוף הטקסט בתצוגה זורמת שאינה צמודה לפריסת הדפסה.
- זיהוי סגנונות Word מתוך `styles.xml`, בעיקר כותרת 1, כותרת 2 וכותרת 3.
- הצגת הערות שוליים בחלונית תחתונה מסונכרנת עם הפסקה הפתוחה בטקסט.
- שינוי גובה חלונית הערות השוליים באמצעות גרירת הידית שמעליה.
- זום משותף לגוף הטקסט ולהערות, או זום נפרד לחלונית ההערות.
- חלון צד נפתח לכל הכלים, כך שאזור העריכה לא מאבד מקום קבוע.
- ניווט לפי כותרות וסליידר קפיצה לאורך המסמך.
- עריכת הטקסט בגוף המסמך ובהערות השוליים.
- שמירה כקובץ DOCX חדש על בסיס הקובץ המקורי.
- שמירה אוטומטית לקובץ המקורי שנפתח, רק אחרי לחיצה על הכפתור.
- שמירת טיוטה מקומית בדפדפן למסמך רגיל שלא נפתח מ־DOCX.
- הורדת תוכן העורך כקובץ טקסט פשוט.
- התקנה כאפליקציה באמצעות manifest ו־service worker.
- טעינת קבצי האפליקציה מהמטמון אחרי פתיחה דרך שרת מקומי או אתר.
- הוספת הערת שוליים מהמקום הנוכחי בעורך באמצעות כפתור, עכבר ימני או `Ctrl+Alt+F`.
- כלים בסיסיים לעריכה בזמן עבודה: הדגשה, כותרות, מספור, תבליטים, יישור וזום.

## הערת שמירה חשובה

בעת שמירת DOCX קיים, האפליקציה משמרת את אריזת קובץ ה־Word המקורית ומעדכנת את טקסטי ה־XML הקיימים. זה נועד לצמצם שינויים במבנה המסמך. סגנונות פסקה קיימים נשמרים, וכאשר בוחרים כותרת 1/2/3 מתוך תיבת הסגנון, האפליקציה מנסה לעדכן גם את `w:pStyle` של הפסקה בקובץ Word.

כדאי לעבוד תמיד על עותק של מסמך חשוב ולבדוק את הקובץ שנשמר ב־Word לפני החלפת המקור.

## שמירה אוטומטית

הכפתור "שמירה אוטומטית" כבוי כברירת מחדל. אחרי פתיחת DOCX דרך כפתור "פתיחת Word" ולחיצה על שמירה אוטומטית, שינויים במסמך יישמרו אוטומטית לקובץ המקורי שנפתח אחרי השהיה קצרה.

התכונה דורשת דפדפן שתומך ב־File System Access API, כמו Chrome או Edge.

## חלונית הערות שוליים

הערות השוליים מוצגות בתחתית חלון העריכה, בדומה לעבודה בתצוגת אינטרנט. בזמן גלילה בגוף המסמך, החלונית מנסה להתמקד בהערה המתאימה לפסקה שנמצאת במרכז אזור העריכה. אפשר גם ללחוץ על מספר הערה בתוך הטקסט כדי לקפוץ אליה בחלונית התחתונה.

## בדיקות

בדיקת הדפדפן יוצרת DOCX קטן, פותחת אותו בעורך, משנה את מבנה הפסקאות, שומרת ובודקת מחדש את ה־XML ואת שימור חלקי האריזה:

```bash
npm install
npx playwright install chromium
npm test
```
